# System Index

> **Layer:** System / Meta-model
>
> This index is the entry-point for the *normative* parts of the system specification.

## Status conventions

- **Normative**: intended to stabilize for v0.1
- **Illustrative**: examples, drafts, stress-tests
- **Exploratory**: ideas that may change significantly

## Directories

- `schemas/` — **Normative**. Formal schemas (YAML for now).
- `specs/` — **Normative**. Prose specifications and rationale.
- `catalogs/` — **Normative**. Enumerated instances (motifs, axes, etc.).
- `bindings/` — **Normative**. Cross-cutting bindings (concept ↔ family/axes/motifs).
